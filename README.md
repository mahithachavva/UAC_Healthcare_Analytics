# Healthcare Capacity Analytics and Forecasting

AI-powered healthcare analytics and forecasting dashboard designed to monitor humanitarian care system capacity, operational load, intake pressure, discharge efficiency, and predictive care trends for the U.S. Department of Health & Human Services.

---

## Project Overview

The Unaccompanied Alien Children (UAC) Program is a federally mandated humanitarian care initiative managed by the U.S. Department of Health and Human Services (HHS).

This project provides a real-time analytics and forecasting platform for monitoring:

- Total healthcare system load
- CBP custody trends
- HHS care capacity
- Intake pressure
- Discharge effectiveness
- Forecasted healthcare demand

The platform transforms raw operational data into actionable insights for healthcare planning and humanitarian response evaluation.

## Dashboard Modules

### 1. System Load Overview
Monitors total operational healthcare burden across CBP and HHS.

### 2. Intake Pressure Analytics
Tracks inflow vs outflow imbalance and backlog accumulation.

### 3. Forecasting Module
Predicts future healthcare system load trends using Prophet forecasting.

### 4. KPI Intelligence
Provides real-time operational metrics for monitoring healthcare system stability.

---

## Forecasting Model

The forecasting engine uses Facebook Prophet for time-series prediction of future healthcare load trends over a 30-day horizon.

Forecasting helps:
- anticipate operational strain
- improve staffing decisions
- optimize care capacity planning

---

## Key Insights

- HHS Care consistently handled the largest healthcare burden.
- Elevated Net Intake Pressure indicated temporary operational strain periods.
- Rolling averages revealed sustained high-load windows.
- Forecasting suggested moderate stabilization in future system demand.
